# Dashboard สารเคมี พร้อมภาพ SDS

แพ็กเกจนี้เพิ่มภาพ SDS จากไฟล์ `สารเคมี.zip` เข้าใน Dashboard เดิม

## ความสามารถใหม่

- ภาพ SDS แยกรายสาร **96 ภาพ**
- พิมพ์ค้นหาชื่อสารเคมีแล้วแสดงการ์ดข้อมูลพร้อมภาพที่จับคู่ได้
- กดชื่อหน่วยงานแล้วแสดงรายการสารเคมีของหน่วยงานนั้น
- กดชื่อสารเคมีหรือภาพเพื่อเปิดหน้าต่างรายละเอียด
- ค้นหาในคลังภาพ SDS ได้ แม้ภาพนั้นยังไม่ได้จับคู่กับรายการในทะเบียน
- รายการที่ชื่อหรือความเข้มข้นไม่ตรงจะไม่ใช้ภาพอื่นแทน

## วิธีอัปโหลดขึ้น GitHub

1. แตกไฟล์ ZIP
2. อัปโหลดไฟล์ทั้งหมดแทนไฟล์เดิมใน Repository
3. ตรวจว่าโฟลเดอร์ `assets/chemical-images` มีภาพ `chemical-001.webp` ถึง `chemical-096.webp`
4. ช่อง Commit message ใส่:

`เพิ่มภาพ SDS และหน้ารายละเอียดสารเคมี`

5. กด Commit changes
6. รอ GitHub Pages อัปเดต แล้วเปิดเว็บไซต์พร้อมกด `Ctrl + F5`

## หมายเหตุเรื่องการจับคู่

ภาพ SDS และทะเบียน Excel ไม่ได้เรียงชื่อสารตรงกันทุกหมายเลข ระบบจึงจับคู่เฉพาะรายการที่ชื่อสารและความเข้มข้นตรงหรือใกล้เคียงอย่างปลอดภัย รายการที่ไม่ตรงจะแสดงข้อความว่า “ยังไม่มีภาพ SDS ที่จับคู่ตรงกับรายการนี้”

คลังภาพ SDS ทั้ง 96 ภาพยังสามารถค้นหาและเปิดดูได้จากช่องค้นหาบน Dashboard


## การแสดงภาพแบบปุ่ม

เวอร์ชันนี้ไม่แสดงภาพ SDS บนการ์ดหรือผลการค้นหาโดยอัตโนมัติ

- กด **ดูรายละเอียด** เพื่ออ่านข้อมูลโดยไม่เปิดภาพ
- กด **ดูภาพ** เพื่อเปิดภาพ SDS ในหน้าต่างใหญ่
- ในตารางทะเบียนมีคอลัมน์ **ภาพ SDS**
- รายการที่ยังไม่มีภาพตรงกันจะแสดงปุ่ม **ไม่มีภาพ** และกดไม่ได้


## เวอร์ชัน Static

แพ็กเกจนี้ไม่มีระบบ Admin และไม่ใช้ Google Apps Script ข้อมูลทั้งหมดอยู่ใน GitHub Repository
