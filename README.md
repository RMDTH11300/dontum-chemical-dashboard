# Dashboard ข้อมูลสารเคมี โรงพยาบาลดอนตูม

## แหล่งข้อมูลล่าสุด

- Excel: `WI-CHEM-001_Quick_Reference_สารเคมี_96_รายการ(3).xlsx`
- ภาพ: `สารเคมี (2).zip`

## สรุป

- สารเคมี: 96 รายการ
- หน่วยงาน: 21 หน่วยงาน
- กลุ่มอันตรายที่มีข้อมูล: 1 กลุ่ม
- ภาพ SDS: 96 ภาพ
- เวอร์ชันระบบ: 1.2.0
- ไม่มีระบบ Admin และไม่ใช้ Google Apps Script

## การจับคู่ภาพ

ระบบจับคู่ตามรหัสแบบ 1 ต่อ 1:

`CHEM-001 ↔ SDS-CHEM-001 ↔ chemical-001.webp`

จนถึง:

`CHEM-096 ↔ SDS-CHEM-096 ↔ chemical-096.webp`

## การปรับข้อมูลเพื่อการแสดงผล

- รวมการสะกด `กลุ่มงานปฐมภูมิ(งานประปา)` เป็น `กลุ่มงานปฐมภูมิ (งานประปา)`
- รวมคำสะกด `โภชนาการการ` เป็น `โภชนาการ`
- CHEM-009 มีคำว่า `กลุ่มงานปฐมภูมิ` อยู่ในช่องกลุ่มอันตราย ระบบย้ายไปแสดงเป็นหน่วยงาน

## วิธีอัปโหลด GitHub

1. แตกไฟล์ ZIP
2. อัปโหลดไฟล์ทั้งหมดแทนไฟล์เดิมใน Repository
3. Commit message:

`ปรับข้อมูลจาก Excel ฉบับ 3 และภาพ SDS ชุดใหม่`

4. รอ GitHub Pages อัปเดต แล้วกด `Ctrl + F5`
