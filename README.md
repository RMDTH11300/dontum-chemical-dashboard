# Dashboard WI-CHEM-001 Quick Reference สารเคมี

ข้อมูลถูกปรับจากไฟล์:

`WI-CHEM-001_Quick_Reference_สารเคมี_96_รายการ.xlsx`

## สรุป

- ข้อมูลสารเคมี: 96 รายการ
- หน่วยงาน: 26 หน่วยงาน
- กลุ่มอันตราย: 19 กลุ่ม
- ภาพ SDS ที่จับคู่และตรวจสอบแล้ว: 17 รายการ
- ไม่มีระบบ Admin และไม่ใช้ Google Apps Script

## รายละเอียดที่แสดง

- Chemical ID และ SDS Code
- ชื่อสารเคมีและหน่วยงาน
- กลุ่มอันตรายเบื้องต้น
- PPE
- Spill Kit / วิธีควบคุม
- ปฐมพยาบาล
- ข้อห้ามสำคัญ
- สถานะทบทวน

## วิธีอัปเดต GitHub

1. แตกไฟล์ ZIP
2. อัปโหลดไฟล์ทั้งหมดแทนไฟล์เดิมใน Repository
3. ช่อง Commit message ใส่:

`ปรับรายละเอียดสารเคมีตาม WI-CHEM-001`

4. กด Commit changes
5. รอ GitHub Pages อัปเดต แล้วกด `Ctrl + F5`

## ข้อควรทราบ

ข้อมูล Quick Reference ใช้เป็นแนวทางตอบสนองเบื้องต้น ไม่ใช้แทน SDS
ฉบับปัจจุบันของผู้ผลิต และควรทบทวนร่วมกับหน่วยงานที่เกี่ยวข้องก่อนอนุมัติใช้


## อัปเดตภาพ SDS เวอร์ชัน 9

- เปลี่ยนภาพทั้งหมดเป็นภาพจาก `สารเคมี_แยกภาพ_รายภาพ(1).zip`
- มีภาพครบ 96 รายการ
- จับคู่ Chemical ID, SDS Code และหมายเลขภาพแบบ 1 ต่อ 1
- ไม่มีการจับคู่ภาพจากชื่อสารเคมี
- ทุกปุ่ม **ดูภาพ** เปิดภาพที่มีรหัสตรงกับรายการนั้น
