# Dashboard ข้อมูลสารเคมี โรงพยาบาลดอนตูม — ชุดข้อมูลใหม่

แหล่งข้อมูล: `รายชื่อสารเคมีในโรงพยาบาลดอนตูม.xlsx`

- ข้อมูลตั้งต้น: 96 รายการ
- หน่วยงาน/สถานะหน่วยงาน: 15 กลุ่ม
- ชื่อสารเคมีไม่ซ้ำ: 96 ชื่อ
- ไม่ระบุหน่วยงาน: 19 รายการ

ระบบประกอบด้วย:

- `index.html` Dashboard สาธารณะ
- `admin.html` เพิ่ม แก้ไข ลบ และแทนที่ข้อมูล
- `apps-script/Code.gs` ฐานข้อมูล Google Apps Script
- `INSTALL_ADMIN.md` คู่มือติดตั้งและเปลี่ยนจากข้อมูลเดิม
- `data/chemicals.js` ข้อมูลตั้งต้นจาก Excel
- `data/chemical_inventory.csv` ข้อมูลรวมแบบ CSV

เมื่อแทนที่ข้อมูลผ่านหน้า Admin ระบบจะสร้างชีตสำรองของข้อมูลเดิมก่อนอัตโนมัติ
