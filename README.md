# Dashboard ข้อมูลสารเคมีภายในโรงพยาบาลดอนตูม

เว็บนี้เป็น GitHub Pages แบบอ่านอย่างเดียว ดึงข้อมูลจาก Google Sheets โดยอัตโนมัติ

## ไฟล์สำคัญ

- `index.html` หน้า Dashboard
- `assets/style.css` รูปแบบหน้าจอ
- `assets/app.js` การอ่านข้อมูล กรอง กราฟ ตาราง และแจ้งวันหมดอายุ
- `assets/config.js` ตั้งค่า Google Sheet และจำนวนวันแจ้งเตือน
- `data/chemicals.csv` ไฟล์สำรองแบบหัวตารางเปล่า

## การตั้งค่าสิทธิ์ Google Sheets

1. เปิด Google Sheet
2. กด **แชร์**
3. ในหัวข้อการเข้าถึงทั่วไป เลือก **ทุกคนที่มีลิงก์**
4. เลือกสิทธิ์ **ผู้มีสิทธิ์ดู**
5. ห้ามตั้งเป็นผู้แก้ไขสำหรับบุคคลทั่วไป

Dashboard จะอ่านข้อมูลผ่าน Google Visualization CSV Endpoint และไม่สามารถแก้ไขข้อมูลในชีตได้

## ชื่อคอลัมน์ที่ระบบรองรับ

ระบบพยายามตรวจหัวตารางภาษาไทยและภาษาอังกฤษอัตโนมัติ เช่น

- หน่วยงาน / แผนก / กลุ่มงาน
- ชื่อสารเคมี / ชื่อผลิตภัณฑ์ / Chemical Name
- CAS No.
- ปริมาณคงเหลือ / จำนวน / Stock
- หน่วย
- ประเภทอันตราย / GHS / Hazard
- สถานที่จัดเก็บ / Storage Location
- วันหมดอายุ / Expiry Date
- SDS / MSDS
- สถานะ
- ผู้รับผิดชอบ

ถ้าหัวตารางไม่ตรง ระบบจะแสดงคำว่า **ไม่พบคอลัมน์** ในส่วน “สถานะการอ่านคอลัมน์” สามารถเพิ่มชื่อหัวตารางใน `ALIASES` ภายใน `assets/app.js`

## เกณฑ์แจ้งเตือนวันหมดอายุ

ค่าเริ่มต้นเตือนล่วงหน้า 90 วัน แก้ได้ใน `assets/config.js`

```javascript
EXPIRY_WARNING_DAYS: 90,
```

## อัปโหลดขึ้น GitHub Pages

1. สร้าง Repository ใหม่ เช่น `dontum-chemical-dashboard`
2. แตกไฟล์ ZIP
3. อัปโหลด `index.html`, โฟลเดอร์ `assets` และโฟลเดอร์ `data` ไปที่หน้าแรกของ Repository
4. กด **Commit changes**
5. ไปที่ **Settings**
6. เลือก **Pages**
7. ในหัวข้อ Build and deployment เลือก:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
8. กด **Save**
9. รอให้ GitHub แสดงลิงก์เว็บไซต์ในหน้า Pages

รูปแบบลิงก์:

`https://ชื่อบัญชี.github.io/dontum-chemical-dashboard/`

## ความปลอดภัยของข้อมูล

GitHub Pages และ Google Sheet ที่แชร์แบบ “ทุกคนที่มีลิงก์” เป็นข้อมูลสาธารณะ ควรตรวจและลบข้อมูลต่อไปนี้ก่อนเผยแพร่:

- ชื่อผู้ป่วย เลข HN เลขบัตรประชาชน และข้อมูลสุขภาพ
- เบอร์โทรศัพท์ส่วนบุคคล
- ชื่อผู้รับผิดชอบที่ไม่จำเป็นต้องเปิดเผย
- ราคาจัดซื้อ เลขเอกสาร หรือข้อมูลภายในที่เป็นความลับ
- รายละเอียดตำแหน่งจัดเก็บที่อาจกระทบความปลอดภัย

หน้า Dashboard ไม่แสดงชื่อผู้รับผิดชอบโดยค่าเริ่มต้น แต่ข้อมูลทุกคอลัมน์ใน Google Sheet ที่แชร์สาธารณะยังสามารถเข้าถึงจากแหล่งต้นทางได้
