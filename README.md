# Dashboard ข้อมูลสารเคมี โรงพยาบาลดอนตูม

แพ็กเกจนี้ใช้ข้อมูลจากไฟล์ Excel:

`รายชื่อสารเคมี  (แยกรายแผนก) (1).xlsx`

ข้อมูลถูกแปลงและฝังไว้ใน `data/chemicals.js` จึงไม่ต้องเชื่อมต่อ Google Sheets และไม่ติดปัญหา CORS

## สรุปข้อมูล

- รายการสารเคมี: 164 รายการ
- หน่วยงาน: 18 หน่วยงาน
- ชื่อสารเคมีไม่ซ้ำ: 129 ชื่อ

ชีตที่นำมาใช้:

- ห้องแล็บ
- ทันตกรรม
- ห้อง x-ray
- OPD
- ห้องยา
- ER
- LR
- Ward ผู้ป่วยใน
- บริหารจัดซื้อ
- ไตเทียม
- โภชนา
- ช่าง
- ผลิต
- จ่ายกลาง,ซักฟอก
- กลุ่มงานปฐมภูมิ,ปะปา
- ห้องSkin
- ฝังเข็ม
- กิจกรรมบำบัด

ไม่ได้นำชีต `หน่วยงาน`, `ตัวอย่างสารเคมี` และ `------` มาใช้ เพราะเป็นชีตประกอบ/ตัวอย่าง/สรุป

## ไฟล์สำคัญ

- `index.html` หน้า Dashboard
- `assets/style.css` รูปแบบหน้าเว็บ
- `assets/app.js` ระบบค้นหา กรอง กราฟ ตาราง และส่งออก CSV
- `data/chemicals.js` ข้อมูลที่ Dashboard ใช้
- `data/chemical_inventory.csv` ข้อมูลรวมแบบ CSV

## วิธีอัปโหลดขึ้น GitHub

1. แตกไฟล์ ZIP
2. เปิด Repository เดิม
3. ลบหรือแทนที่ไฟล์เดิมทั้งหมดด้วยไฟล์ในแพ็กเกจนี้
4. ตรวจว่า `index.html`, `assets` และ `data` อยู่หน้าแรกของ Repository
5. ช่อง Commit message ใส่:

`ปรับ Dashboard ให้ใช้ข้อมูลสารเคมีจากไฟล์ Excel`

6. กด **Commit changes**
7. เปิดหน้าเว็บไซต์แล้วกด `Ctrl + F5`

หากยังไม่ได้เปิด GitHub Pages:

1. ไปที่ **Settings**
2. เลือก **Pages**
3. Source เลือก **Deploy from a branch**
4. Branch เลือก **main**
5. Folder เลือก **/(root)**
6. กด **Save**

## การอัปเดตข้อมูลในอนาคต

เมื่อไฟล์ Excel มีการแก้ไข ต้องสร้าง `data/chemicals.js` และ `data/chemical_inventory.csv` ใหม่ เพราะ Dashboard เวอร์ชันนี้ใช้ข้อมูลแบบฝังในเว็บไซต์ ไม่ได้อ่าน Excel โดยตรงจากเบราว์เซอร์

## หมายเหตุข้อมูล

ระบบใช้โครงสร้างดังนี้:

- A: Code สารเคมี
- B: ชื่อสารเคมี
- C: Trade Name
- D: ปริมาณที่มีเก็บไว้
- G หรือ E: สถานที่เก็บ
- H หรือ F: การใช้ประโยชน์
- I–R: ความเป็นอันตราย
- S–W: อุปกรณ์ป้องกันอันตรายส่วนบุคคล
